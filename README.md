# 🎮 RG Gamers — Free PC Game Downloads

A dark-themed gaming website to browse and download PC games in WinRAR format via Pixeldrain, MEGA, GoFile, and more.

**Owner:** Rohit Gaikwad

---

## 🚀 Deploy on GitHub Pages

1. Upload all files to your GitHub repository root.
2. Go to **Settings → Pages** → Source: **main branch / root**.
3. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

## 📁 File Structure (Flat — all files at root)

```
/
├── index.html          ← Home page
├── games.html          ← All games list
├── game-detail.html    ← Game detail page
├── categories.html     ← Browse by genre
├── about.html          ← About page
├── contact.html        ← Contact form
├── admin.html          ← Admin panel (password: RGAdmin2025)
├── style.css           ← All styles
├── main.js             ← All JavaScript
└── images/
    └── placeholder.jpg ← Fallback cover image
```

## 🔐 Admin Panel

- URL: `/admin.html`
- Default password: `RGAdmin2025`
- Change the password in `main.js` → `ADMIN_PASSWORD` constant

## ☁️ Supported Download Hosts

- Pixeldrain · MEGA · GoFile · MediaFire · WeTransfer · Catbox · Imgur
